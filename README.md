# Data_Analytics_Bootcamp_Labs  

Practice labs and exercises from the Ironhack Data Analytics Bootcamp  


This repository contains a collection of hands-on labs and exercises completed during the Ironhack Data Analytics Bootcamp.  
The labs cover core data analytics skills including Python programming, SQL querying, data cleaning, exploratory data analysis, statistics, and introductory machine learning.  

These exercises were originally created as individual repositories during the bootcamp and have been consolidated here for better organization.  


# Technologies Used  

- Python
- Pandas & NumPy
- SQL
- Data Cleaning & Wrangling
- Exploratory Data Analysis (EDA)
- Statistics
- Machine Learning fundamentals
- Jupyter Notebook
